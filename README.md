## ðŸ§  Methodology
1. Created a dataset of customer reviews
2. Used TextBlob to calculate sentiment polarity
3. Classified sentiment based on polarity score:
   - Positive (polarity > 0)

4. Stored results in a Pandas DataFrame
5. Visualized sentiment distribution using a bar chart
